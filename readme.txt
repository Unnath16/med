for this project we need
1. Mysql server:
i have used a xamp server which has phpmyadmin
2. any text document viewer for the .html page to be viewed
3. Latest netbeans which supports web pages


step 1:
Open the project in  Netbeans 
 
Step2:
Open and run the database in phpmyadmin


Step3:
Run the project to create a session


Step4:
Open the index.html file to check the website
Create a user by signup option in customer and login.


Please Note: No other pages will work if not logged in. This is because each page is handled by Session.
